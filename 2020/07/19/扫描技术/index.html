扫描技术

==手工与工具相结合==

一.主机探测:确定目标主机是否存活

二.端口扫描:寻找在线主机所开放的端口，在端口上运行的服务，甚至操作系统类型和更详细信息

三.基础知识

常见端口号

  1. 22 SSH 1433 SQL Server
    21 ftp 443 HTTPS
    23 Telnet 1521 Oracle
    25 SMTP 3306 MySql
    80 HTTP 3389 RDP

  2. 工具Nmap

    -重要常用参数

    -O 扫描目标IP系统信息
    -p 指定端口号
    -sP ping扫描
    -sV 服务版本探测
    -A 全面扫描
    -oN 保存txt

  3. 工具Hydra

    1. hydra.exe -l user -P 密码字典路径 ip + 协议
    2. hydra.exe -L用户字典路径 -P 密码字典路径 ip + 协议
